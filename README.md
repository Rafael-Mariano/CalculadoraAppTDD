# CalculadoraApp
## Objetivo

<p>O projeto CalculadoraApp foi desenvolvido com o objetivo de treinar o método Test-Driven Development (TDD).</p>
<p>O modelo tem por objetivo iniciar os testes de funcionalidade do projeto antes de realizar a implementação do código principal.</p>
<p>Ao executar o arquivo teste pela primeira vez, todas as funcionalidades irão retornar com erro, pois não há nada implantado.</p>
<p>Depois da implantação das funcionalidades no projeto, o arquivo de teste irá retornar com sucesso os casos.</p>
<p>Na sequência vem a questão da refatoração, aplicando as boas práticas de programação.</p>

## Funcionalidades
<div>
  <p>A calculadora possúi as funcionalidades básicas: somar, subtrair, multiplicar e dividir. Além disso há a opção do histórico, que busca apresentar os
  ultimos 3 resultados que foram utilizados.</p>
  <p>Esta sendo utilizado apenas números inteiros, isso pode influenciar nos resultados das divisões.</p>
  <p>O projeto foi desenvolvido apenas em console</p>
</div>
